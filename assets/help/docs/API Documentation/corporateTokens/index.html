Corporates Token

This API allows publisher/partner to generate corporate tokens by using the parameters mentioned below: 

  {

}

Request URL

POST: https://dev-us-lms-backendapi.plaza-network.com/api/fe/v1/admission/corporateTokens

Parameters Description

Parameter Type Mandatory* Description
clientID string Mandatory*
