POST: https://dev-us-lms-backendapi.plaza-network.com/api/fe/Coupon/GetCouponCode

Parameters Description

Parameter Type Mandatory* Description
publicKey String Mandatory*
Token string Mandatory* This is token which is generated when user login into system.
ClientCode String Mandatory*
CardCode string Mandatory*
ServiceCode string Mandatory*
ProductCode string Mandatory* This is unique code of product that user uses.
TransactionDateTime Date Mandatory* This is the transaction date and time when user make payment.
TransactionUTCDateTime Date Mandatory* This is the transaction UTC date and time when user make payment.
OrderID Number Mandatory* This is unique order id of user's order.
DeviceID Number Mandatory*
PartnerCode String Mandatory*
ItemDescription string Mandatory* Item that user odered.
PromotionCode string Mandatory* This is promotion code which user can apply for discount.
ReceiptNumber string Mandatory* This is receipt number of user's bill.
Remarks string Optional* This is field where user can give remarks about service or product.
NotificationMode Option Mandatory* This is notification mode in which user will notified.
NotificationID string Mandatory*
LastName string Mandatory* This is user's last name.
FirstName string Mandatory* This is user's first name.
MiddleName string Optional* This is user's middle name.
