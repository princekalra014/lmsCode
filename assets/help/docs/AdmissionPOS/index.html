Admission POS

In 'Admission POS'screen user can see all the admissions done in the outlet with its in-depth details.

"Admissions" tab is accessible from the sidebar. Click on it to get an in-depth data view of the admissions.

User can search for the admission(s) by entering its details in the search field.

Or, User can click on the dropdown icon for advanced search in which options are present as different fields, use those to filter the search.

Click on "Reset" button to reset the form values.

Click on "Search" button to search for the result.

User can find the Admissions list in the table.

You can view admission's details by double clicking on that row of the table for whose details you want to see.

System then populates a screen where you can see the complete detail of that particular order including admission number, member ID, flight no, airlines, outlet name, etc.

Click on 'New' button to create an admission.

A list of 'Token Types' will populate. Select a token type e.g. Qr Code.

Focus will bring to the 'Token Code' field. Enter the token code.

After selecting the entitlements using the checkbox, click on 'Verify Token Entitlements'.

Click on the 'Collect payment button'. To collect the payment and proceed.

It is optional to fill the Guest Name, Email & Phone number. Click on 'New order' to discard this transaction.

Click on 'Create Order & Print Receipt' to create the order and print the receipt.

Click on 'Create Order' to Create an order without printing its receipt.

Click on the 'Export' button to get details about the admission in an excel sheet.
