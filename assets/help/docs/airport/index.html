Port/Station

You can add multiple port/station directly by importing complete file containing list of port/station and their details.

"Port/Station" tab is accessiable from the sidebar as a "Masters" submenu item. Click on it to get in-depth data view of all the ports/stations.

You can search for the port(s)/station(s) by entering its details in the search field.

Or, you can click on dropdown icon for advance search in which you are having more option to filter out list.

Click on "Reset" button to reset the form values.

Click on "Search" button to search for the result.

You can find port/station list in the table.

You can view port's details by double click on that row of the table for whose details you want to see.

System then populates a screen where you can see the complete detail of that particular port including port name, port code, port city, port state, etc.

Click on "Reset" button to reset the fields.

Click on submit button

You can also add new port by clicking on "New" button present on the top right corner on the main screen.

Fill the following information to add new port in the system.And click on submit .

Click on "Import" button so that you can import the complete file having required data in one go.

Follow the steps present in "Import" tab present the sidenav of this documentation.

Detailed view of data of this screen can be exported by clicking on the Export button placed at the top right corner of the screen. It downloads the details in the form of an excel sheet for further use.

Below are the input fields used in this screen:

Name of the field Entry Mode Mandatory Comments
Port Name Free text with dropdown Yes You have to enter starting three alphabets of the port name and it will show you the result based on that. Once you select port name it will auto fill some fields like port code, port city, port state etc. You have to fill that too.
Port Code Free text Yes System generated once you select port name. Also, you can fill it yourself. Code should be in captial letters always.
Port City Free text Yes System generated once you select port name. Also, you can fill it
Port State Free text with dropdown Yes System generated once you select port name. Also, you can fill it. You have to enter the state in which this port exists.
Port Country Free text Yes System generated once you select port name. Also, you can fill it
Port Continent Free text Yes You have to enter the continent in which this port exists.
Port Region Free text Yes You have to enter the Region of the port.
Port Timezone Free text Yes You have to enter the timezone of the port.
Port Timezone (In UTC) Free text Yes You have to enter the timezone (In UTC) of the port.
Port Type Selection Yes Port Type could have the following values :- Airport and Railway.
Is Active Switch (Toggle) Button Yes Yes/No
