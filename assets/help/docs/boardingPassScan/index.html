Request URL

POST: https://dev-us-lms-backendapi.plaza-network.com/api/fe/boardingPass/save

Parameters Description

Parameter Type Mandatory* Description
isManual Boolean Optional*
createTime Date Mandatory* This is date and time of creation of boarding pass.
flightFrom string Optional* This is country code from where flight departure.
flightTo string Optional* This is city code where flight landed.
fromCity string Optional* This is city code where flight landed.
toCity string Optional* This is city code where flight landed.
airlineCode string Optional* This is airline code under which that flight belongs.
flightNumber Number Optional* This is a unique number provided to every flight.
classCode string Optional* This is the code of class in which user travel in flight
dateOfFlight Date Optional* This is the date of journey.
seatNumber string Optional* This is seat number aloted to user.
airlineNumericCode Number Optional* This is airline code in numbers.

Success:
