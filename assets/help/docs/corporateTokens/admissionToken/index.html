Admission Instruction

You can add new tokens manually or import complete file containing list of Admission Instructions and their details. Also, edit existing Admission Instruction's details in the system for futher use.

Admission Instruction Types

Credit Card
To get airport lounge access customers can bring a credit card that comes with a lounge access feature or lounge access can be at discounted rate .The number of free airport lounge access varies from card to card. Few premium credit cards offer unlimited access to airport lounges along with complimentary guest.click here.

QR Code
To get airport lounge access customers can bring some vouchers or QR-Codes issued by third party. To set up such Admission Instruction click here

Airline Invitation
Airline Passengers can get complimentary or discounted lounge access or other facilities, depending upon the Airline Program. The Admission Instruction will be setup by using the boarding pass. click here.

Airline Staff
Airline Crew Members gets complimentry lounge access on the airports .They need to bring the crew ID passes to get lounge access. To set up Airline Staff Token click here.

Smart Traveller Voucher
This is a loyality program run by Premium Plaza for members of smart traveller app. ST app API is called to know the membership ID validation. Arrture points can be used to redeem for various rewards.

Corporate Client Membership
Corporate Client Membership : Corporate Partners provides Coupon/Vouchers/Memberships to their customers, they can access complimentary or discounted lounge access and other facilities on the airport.

Walk-in
A customer can avail lounge access or other facilities without any credit card or coupon.

Entitlement Tab :

Applicable To

selection

Yes

This field displays the token is applicable to which category of guests. 

Select from the following Values:-Primary, Accompanying Guest, Child, Fallback, Product, Group

Entitled product

Free text with dropdown

Yes

This field displays the product(s) which is/are bind to the outlet where the Admission Instruction is setup. Enter first three letters of the product, if it is bind to the outlet in dropdown suggestion will be shown and you will be able to select the product. Pre requite is that the product should be binded to the outlet if not then go to "Product Available At" tab in product screen and make the product available.

Entitled Quantity

Number

Yes

This is a numeric field, number entered in this field will be multiplied with the binded product's service time and will eventually denote the duration of the admission. eg "Lounge Usage" is bind to a token and 2 is entered in this field then the duration of this Admission Instruction will be 2*1 hour service time = 2 hours

Quota

Checkbox/Number

Yes

This is field displays the number of time the guest can avail the binded product. Click on checkbox in front of Unlimited if the quota is to be set as unlimited or else select the input the number value. 

Within

Selection

Yes

This field displays for how long the quota is valid for. Select a duration from the selection box.

Discount Type

Selection

Yes

This field has 2 selections, Absolute and Percentage. If a specific price of the product has to be setup on a token then absolute will be selected and in Discount Price field the price will be given. Else percent will be selected and value of percentage discount will be entered in Discount Price field. 

Currency

Selection

Yes

This is a Selection field where from the menu you can select the currency name which you want to bind on the Admission Instruction

Discounted percent

Number

Yes

Refer above described Discount Type field. Eg on  a product the token is entitled to have 20% discount then in Discount Type percent will be selected and in Discounted Price 20 will be selected.

Free text

No

Eg Amex has benefit IDs which defines the entitlements of their customers to map those entitlements we bind the benefit id to MAS 3.0’s entitlements.
