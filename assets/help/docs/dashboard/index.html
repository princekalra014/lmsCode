Dashboard

In the 'Dashboard' section of LMS 3.0 users can have an overview like of the admission done e.g. which Partner's guests came to the outlet, How many orders were created, How many products were sold, How many guests are present at a particular time etc.

"Dashbaord" tab is open by default once you logged in the system. Else, it is accessible from the sidebar. Click on 'Dashboard' to get an in-depth data view of the entire system.

In the top left section, you can see the count of the users, partners, orders and product, no. of guests etc. You can filter out the data by time given in dropdown like last week, last month, last year, all time.

On the right of it, you can see the Number of guests in the outlet at the moment and the count of Admission Instructions using which guests have entered the outlet.

In "Today's Count in Outlet" you can see the bar graph which denotes total number of guests in the outlet on hourly basis.

In 'Outlet Capacity Utilised' User can see how much percentage an outlet is full or vacant at the moment. This gives an overview of occupancy of the outlet.

“Top Sales In Amount” shows the top sales on hourly basis of an outlet in the form of a bar graph.

'Top 3 Products' section shows the most availaled services/products in the system.

In "Visit by Admission Instruction Name" a pie chart illustrates which type of admission instruction is used by the guest to take admission e.g. Walk-in, Credit card, QR Code etc. it also denotes the number of guests admitted using each type of instructions user just needs to hover the pie chart.

"Visit by Partner Name" section notifies which type of partners had how many guests in the outlet over the selected duration by hovering the pie chart.
