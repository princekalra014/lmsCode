Get Outlets

This API allows publisher/partner to get outlets by using the parameters mentioned below: 

    {
      "search":"Hong Kong",
      "modifiedFrom": "2022-03-29",
      "modifiedTo": "2022-06-01"
     }

Request URL

POST: https://dev-us-lms-backendapi.plaza-network.com/api/fe/v1/plaza/lms/getOutlets

Parameters Description

Parameter Type Mandatory* Description
search String Mandatory* User have to enter Outlet name to access specific outlet data.
modifiedFrom Date Mandatory* Help the user to know about the status of the order starting from the mentioned date.
modifiedTo Date Mandatory* Help the user to know about the status of the order till the mentioned date.
