Get Token

This API allows publisher/partner to generate token while login by using the parameters mentioned below: 

  {
    "username": "aman.bansal",
    "password": "IshAAn2901$"
}

Request URL

POST: https://dev-us-lms-backendapi.plaza-network.com/api/fe/Authenticate/GetToken

Parameters Description

Parameter Type Mandatory* Description
username string Mandatory* This is user's unique name created when they signup.
password String Mandatory* User created this at the time of signup to secure there account from others.
