Get Admissions

This API allows publisher/partner to get admission by using the parameters mentioned below: 

  {
      "partnerName":"PPG-Plaza",
      "fromDate": "2022-12-07",
      "toDate": "2022-12-08"
  }

Request URL

POST: https://dev-us-lms-backendapi.plaza-network.com/api/fe/v1/plaza/lms/getAdmissions

Parameters Description

Parameter Type Mandatory* Description
partnerName String Mandatory* User needs to enter the name of the partner (for whom the admission details are to be fetched) from the Partner screen.
fromDate Date Mandatory* User needs to enter the date from which the user wants to fetch the admission details.
toDate Date Mandatory* User needs to enter the date from which the user wants to fetch the admission details.
