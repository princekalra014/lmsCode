Module

User can see all modules present in LMS 3.0 and add new module details manually as well. Also, edit existing module's details in the system for further use.

"Module" tab is accessible from the sidebar as a "Masters" submenu item. Click on it to get an in-depth data view of all the Modules.

User can search for the module(s) by entering its details in the search field.

Or, User can click on the dropdown icon for advanced search in which options are present as different fields, use those to filter the search.

Click on "Reset" button to reset the form values.

Click on the "Search" button to search for the result.

User can find the list of modules in the table.

User can edit the particular module detail from the system by double clicking on the respective row of the table.

System then populates a screen where the user can see the complete detail of that particular action including module name, module short name, module description, Help URL. You can edit any field by clicking on them.

User can save the changes by clicking on the “Submit” button or you can reset the changes by clicking on the “Reset” button.

Also, user can see one horizontal tab namely, Module Action below the form fields.

User can edit the particular module action from the system by double clicking on the respective row of the table.

The system will populate the new screen for you, where you can view/edit the existing information. Click on "Reset" button to reset the edited fields. Click on "Add Action" to save the edited fields changes.

User can also add a new module action by clicking on the "Add Action" button present on the top of the table.

In this screen,user can fill in the following details to add new action in the system. Click on "Reset" button to reset the fields. Click on "Add Action" to save the data.

User can also add a new Module by clicking on the “New” button present on the top right corner on the main screen.

Fill the following information to add new module in the system. User can click on "Reset" button to reset the form and can click on the "Submit" button to save the filled details in the system.

Detailed view of data of this screen can be exported by clicking on the Export button placed at the top right corner of the screen. It downloads the details in the form of an excel sheet for further use.

Below are the input fields used in this screen:

Name of the field Entry Mode Mandatory Comments
Module Name Free text with dropdown Yes Users have to enter the first three alphabets of the module name and system will show the result based on the input. Once user selects a module name it will auto-populate the fields like Module Short Name, Module Description, Help Url.
Module Short Name Free text Yes This field is system generated once the user selects module name module Short name is generated by the system. Also, user can enter the short name( should be in capital letters always).
Module Description Free text Yes System generated once you select module name. Also, you can fill it
Help Url Free text Yes This field is system generated once the user selects module name description is auto-populated. Also, user can enter it manually.
