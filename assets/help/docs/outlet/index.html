Outlet

You can add new partners manually or import complete file containing list of outlets and their details. Also, edit existing outlet's details in the system for futher use.

Process to import the Outlets

"Outlet" tab is accessiable from the sidebar as a "Masters" submenu item. Click on it to get in-depth data view of all the outlets.

You can search for the outlet(s) by entering its details in the search field.

Or, you can click on dropdown icon for advance serach in which you are having more option to filter out list.

Click on "Reset" button to reset the form values.

Click on "Search" button to search for the result.

You can find outlets list in the table.

You can view outlet's details by double click on that row of the table for whose details you want to see.

System then populates a screen where you can see the complete detail of that particular outlet including outlet name, type, category, address, etc.

Also, you can see four horizontal tabs in the details page below the form fields - Outlet Photos, Users, Outlet Timings, Localisation.

By default "Outlet Photos" data is visible, where you can see the photos of the outlet or can add new ones.

You can see the available photos of the outlet in the system.

You can also add new photos by clicking on "Add New Outlet Photos" button present on the top of the outlet images.

In this screen, you can upload the photos of the outlet.

Click on "Reset" button to delete the uploaded photos.

Click on "Add Outlet Photos" to save the uploaded photos.

Click on "Users" tab to view, edit or new add users for the outlet.

You can find the table of existing users.

Click on "Outlet Timings" tab to view, edit or new timings for the outlet.

You can find the table of currently running timings for the outlet.

You can also add new timings by clicking on "Add New Timings" button present on the top of the table.

In this screen, you can fill the following details like a switch button for every day which represents on which the outlet is closed/open and timings (opening and closing) for each day specifically for the outlet.

You can also add multiple timings for single day by clicking on "Add Hours" link.

It will open another field each time you click on "Add Hours" option.

Click on "Reset" button to reset the fields.

Click on "Add Timings" to save the data.

Click on "Localisation" tab to view the data in different languages.

You can find the table for existing attributes.

Click on "Save" to save the data.

Fill the partner name and Partner Outlet ID As shown below and click on "Add Partner Link".

Click on "Outlet Settings" button to view/edit the settings.

The popup will open for you where you can view and add the outlet settings by clicking on add outlet setting.

Fill all the details here.

Click on "Save" button to save the save the outlet settings.

Clicking on this button will download all the holistic view admission tokens associated with the outlet in pdf format provided the outlet has some admission token binded to.

Clicking on the submit button will bring the user to the outlet screen.

You can also add new outlet by clicking on "New" button present on the top right corner on the main screen.

Fill the following information to add new outlet in the system.

Click on "Reset" button to reset the form.

Click on "Save" button to save the filled details in the system.

You can import one or more outlets by clicking on this button. An excel template is downloaded, open the template input the outlet details and upload the template. All the imported outlets will be fetched in the outlet screen.

Follow the steps present in "Import" tab present the sidenav of this documentation.

Below are the input fields used in this screen:

Name of the field Entry Mode Mandatory Comments
Outlet Name Free text with dropdown Yes You have to enter starting three alphabets of the outlet name and it will show you the result based on that. Once you have selected any outlet from the dropdown, you will see some of the fields get auto filled.
Outlet Category Selection Yes Category could have following values - Arrival, Departure, Outside Airport
Outlet Code Free Text Yes You have to enter the shortcode of that outlet which must be unique.
Outlet Type Selection Yes Type could have following values - Dinning, Kids Area, Lounge, MASSAGE CENTRE, Meeting Room, SPA center
Outlet Brand Free Text Yes This field displays the information about the brand that owns the outlet.
Outlet Owned By Free Text with dropdown Yes Name of the owner of the outlet and you have to enter the first three letters of the name and select it from dropdown.
Outlet Operated By Free text with dropdown Yes Name of the person who is operating things in that outlet.First it needs to be defined in vendor screen.
Parent Lounge View only No System generated
Business Line Selection Yes Business Line could have following values - Lounge, Spa, F&B
Outlet Receipt Free Text with Dropdown Yes First its needs to be Created in Receipt Template Screen. Here Just type first 3 alphabets and select that receipt name from dropdown
Address Line 1 Free text Yes Enter the address of the outlet
Address Line 2 Free text No Enter the address of the outlet
Port Free text with dropdown Yes First its needs to be defined in Port/Station Screen.You have to enter starting three alphabets of the port and it will show you the result based on that. If you hover over on "i" symbol you will get the information about that port.
Journey Type Selection Yes Journey could have following values -Both, Domestic, International
Travel Direction Selection Yes Travel Direction could have following values - Arrival, Both, Departure
Terminal Code Free text Yes From Airport Configuration
Sub Termial Free text No If the terminal has multiple locations, then either use the gate code, or any other indication to reflect sub location within the terminal.
Zone Selection Yes Zone could have following values - Before Security, After Security, In Transit
Capacity Number Yes Number of people allowed (sitting area) at a time in outlet
IsActive Switch (Toggle) button Yes Yes/No
Is Secured Switch (Toggle) button Yes Yes/No
Secured By Selection Yes Secured By could have following values - Keycode, None, Queue, eGate
Base Currency Selection Yes ISO Code of currency. Currently currency could have following values - USD, YEN etc.
Date of Opening Date Yes Date of opening of outlet
Phone No Number Yes Phone number of the outlet
Fax Free text No Fax number of the outlet
Outlet Description Free text Yes Enter description if you want to add for the outlet
Outlet Photo Upload No You can upload multiple photos of the outlet
Users View only No Once the outlet is created, you can add that outlet in user screen for that particular user and once its done, Here it will be visible.
Outlet Timings Switch(toggle) button and time No You can close/open day for the outlet by clicking on switch button. And add multiple opening/closing timings for the outlet
Localisation Free text No You can add attributes in different languages
Outlet Partner Link Free text No This tab has two fields Partner Name and Partner Outlet I'd. In partner field we enter the name of the partner and in Partner Outlet we enter the I'd provided by the partner.
Partner outlet I'd : In this field we input the I'd of the outlet which is provided by the partner. This I'd syncs the outlets eg Partner A give I'd of Outlet A (in MAS 3.0) as 1024. Then for outlet A in outlet partner I'd we input 1024.
