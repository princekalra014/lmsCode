Overview

Welcome to PPG LMS. This is the quick overview of the system that gives the arial view of the features.

Just after you login there is the dashboard screen that gives the overall status of the system.

On Dashboard you can see some key stats such as number of users, partners, orders, and products etc. You can also see the current count of members and Admission Instructions in the lounge.

Dashboard have some stats with visual representation of data in forms of ghraphs and charts. Some of these are: Today's count in lounge, Top selling products, Total sales, Visit by Admission Instruction name, and Visit by partner name etc. At some places the data is grouped while visualised for better understanding of the ratio. You can filter these stats with dropdowns having options for last week, month, year and all time.

On the left side navbar, there are many items that can help you to navigate to specific sreens to see and manage things in detail.

In admission, see the list of admissions for the lounge service. Each entry has some key information attached to it.

Orders section contains the list of orders received for a lounge. More details can be viewed for a particular order.

Partner responses contains responses list from partners for a given outlet. Every response has some metadata associated with it.

Adding each record to the system can be very time consuing at time. To save time, go to File Import/Export section and import data from sheets with just few click. You can export the data out if needed.

Masters section has a sub-menu which contains many options that you can manage. These are Partners, Outlets, Products, Admission Instructions, Partner Contracts, Airports, Users, Members, and Devices. You can list, manage or add new entries for these masters. This is just an overview of what you can manage with PPG LMS. For more details about each section, please refer to the related documentation page.
