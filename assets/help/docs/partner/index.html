Partner

You can onboard new partners manually or import complete file containing list of partners and their details. Also, edit existing partner's details in the system for futher use.

How to Import Partners

"Partner" tab is accessiable from the sidebar as a "Masters" submenu item. Click on it to get in-depth data view of all the partners.

You can search for the partner(s) by entering its details in the search field.

Or, you can click on dropdown icon for advance serach in which you are having more option to filter out list.

Click on "Reset" button to reset the form values.

Click on "Search" button to search for the result.

You can find partners list in the table.

You can edit/delete the particular partner from the system by clicking on the respective icons present in the "Action" column of the table.

If you have clicked on delete(trash) icon then it will open the popup for you, where you have to click on "Yes' button if you are sure about deleting. Else, click on the "No" button.

If you have clicked on edit icon then you can see the details of that particular partner. Or, you can double click on the row of the table to view/edit particular partners information.

System then populates a screen where you can see the complete detail of that particular partner including partner name, type, category, region, etc.

Also, you can see two horizontal tabs in the details page below the form fields - Partner Entities and Contacts.

By default "Partner Entities" data is visible, where you can see the table of partner entities.

You can edit/delete the particular entity from the system by clicking on the respective icons present in the "Action" column of the table.

If you have clicked on delete(trash) icon then it will open the popup for you, where you have to click on "Yes' button if you are sure about deleting. Else, click on the "No" button.

If you have clicked on edit icon then you can see the details of that particular entity. Or, you can double click on the row of the table to view/edit particular entities information.

The system will populate the new screen for you, where you can view/edit the existing information.

Click on "Reset" button to reset the edited fields.

Click on "Add Entity" to save the edited fields changes.

You can also add new entity by clicking on "Add Entity" button present on the top of the table.

In this screen, you can fill the following details to add new entity in the system.

Click on "Reset" button to reset the fields.

Click on "Add Entity" to save the data.

Click on "Contacts" tab to view partners contact information.

You can edit/delete the particular contact from the system by clicking on the respective icons present in the "Action" column of the table.

If you have clicked on delete(trash) icon then it will open the popup for you, where you have to click on "Yes' button if you are sure about deleting. Else, click on the "No" button.

If you have clicked on edit icon then you can see the details of that particular contact. Or, you can double click on the row of the table to view/edit particular contact information.

The system will populate the new screen for you, where you can view/edit the existing information.

Click on "Reset" button to reset the edited fields.

Click on "Add Contact" to save the edited fields changes.

You can also add new contact by clicking on "Add New Contact" button present on the top of the table.

In this screen, you can fill the following details to add new contact in the system.

Click on "Reset" button to reset the fields.

Click on "Add Contact" to save the data.

You can also add new partner by clicking on "New" button present on the top right corner on the main screen.

Fill the following information to add new partner in the system.

Click on "Reset" button to reset the form.

Click on "Save" button to save the filled details in the system.

Click on "Partner Settings" button to view/edit the settings.

The popup will open for you where you can view/edit the partners settings.

Click on "Close" button to close the popup.

Click on "Save" button to save the save the edited information.

Click on "Import" button so that you can import the complete file having required data in one go.

Follow the steps present in "Import" tab present the sidenav of this documentation.

Detailed view of data of this screen can be exported by clicking on the Export button placed at the top right corner of the screen. It downloads the details in the form of an excel sheet for further use.

Name of the field Entry Mode Mandatory Comments
Partner Name Free text with dropdown Yes Enter the name of the partner you want to create in this field. Country Selection Yes Zone could have following values - United States of America, Hong Kong etc.
