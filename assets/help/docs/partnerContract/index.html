Partner Contract

You can add new contract manually or import complete file containing list of contracts and their details. Also, edit existing contract's details in the system for futher use.

"Partner Contract" tab is accessiable from the sidebar as a "Masters" submenu item. Click on it to get in-depth data view of all the contracts.

You can search for the contract(s) by entering its details in the search field.

Or, you can click on dropdown icon for advance serach in which you are having more option to filter out list.

Click on "Reset" button to reset the form values.

Click on "Search" button to search for the result.

You can find contracts list in the table.

You can view contract's details by double click on that row of the table for whose details you want to see.

System then populates a screen where you can see the complete detail of that particular contract including partner name, tax identification number, partner type, contract type, date, etc.

If quota is defined on partner contract level then then this field displays if the defined quota is refreshed on daily/ monthly/ yearly basis.

The quota in this field displays the total quota of all the tokens associated to this partner contract. If the total quota of tokens is 10 then firstly uncheck the checkbox against unlimited then a field will appear enter '10' in the field. This is how you bind combined quota for more than one token on partner contract level

Also, you can see one horizontal tab in the details page below the form fields - Contract Tokens.

By default "Contract Tokens" data is visible, where you can see available tokens for that contract.

You can see the available contract token details and other information in the table.

You can also add new token for for the contract by clicking on "Add Token" button present on the top right of the table.

In this screen, you can check the checkbox(es) to select tokens at a time.

Click on "Reset" button to reset the edited information.

Click on "Add Token" button to save the token details.

Click on "Submit" button to submit created partner contract.

the status of submited partner contract should be "Approval pending" as shown below

To reject submitted partner contract click on "Reject" button and mention the reason in pop up window.

To approve submitted partner contract click on "Approve" button

The status of approved contract should be "Approved" As shown below

Click on "Action History" button.

whatever action is performed on a particular contract, will get logged into action history

To renew the Partner Contract click on Renew Contract button.

It will open the popup for user, where you have to click on "Yes' button if you want to renew contract. Else, click on the "No" button.

To add tokens in approved Partner contract then click on "Add Token" button.

Select the tokens which you want to add and click on update contract button.

Partner contract will get updated.

To deactivate the created partner contract click on "Deactivate" button

Mention the reason and comment for that partner contract then click on "Deactive" button

The status of deactivated partner contract should be "Deactivated"

You can also add new contract by clicking on "New" button present on the top right corner on the main screen.

Fill the following information to add new contract in the system.

Click on "Reset" button to reset the form.

Click on "Save" button to save the filled details in the system.

Click on "Import" button so that you can import the complete file having required data in one go.

Follow the steps present in "Import" tab present the sidenav of this documentation.

Detailed view of data of this screen can be exported by clicking on the Export button placed at the top right corner of the screen. It downloads the details in the form of an excel sheet for further use.

Below are the input fields used in this screen:

Name of the field Entry Mode Mandatory Comments
Partner Name Free text with dropdown Yes You have to enter starting three alphabets of the partner name and it will show you the result based on that, select it from dropdown.
Tax Identification Number View Only - System generated
Partner Type View only - System Generated
Contract Starts On Date Yes Enter/select the date on which contract will start
Contract Ends On Date Yes Enter/select the date on which contract will end
Contract Type Selection Yes Business Line could have following values - Global, Local, Regional
Status View only - System generated
Last Modified On View only - System generated
Last Modified By View only - System generated
Contract Quota Refresh - Yes If quota is defined on partner contract level then then this field displays if the defined quota is refreshed on daily/ monthly/ yearly basis.
Contract Quota (All Entitlements) Checkbox/Number Yes The quota in this field displays the total quota of all the tokens associated to this partner contract. So you can either select "unlimited" Checkbox or you can Enter the number and increase or decrease the values.
