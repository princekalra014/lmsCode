Pincode

You can onboard new pincode manually or import complete file containing list of pincode and their details. Also, edit existing pincode details in the system for futher use.

"Pincode" tab is accessiable from the sidebar as a "Masters" submenu item. Click on it to get in-depth data view of all the pincode.

You can search for the pincode by entering its details in the search field.

Or, you can click on dropdown icon for advance serach in which you are having more option to filter out list.

Click on "Reset" button to reset the form values.

Click on "Search" button to search for the result.

You can find pincode list in the table.

You can edit the particular pincode details from the system by double click on the row .

System then populates a screen where you can see the complete detail of that particular pincode including pincode , country name,country ISO code, state name, city name, region.

Click on "Reset" button to reset the form.

Click on "Save" button to save the filled details in the system.

You can also add new pincode by clicking on "New" button present on the top right corner on the main screen.

Fill the following information to add new pincode in the system.

Click on "Reset" button to reset the form.

Click on "Save" button to save the filled details in the system.

Click on "Import" button so that you can import the complete file having required data in one go.

Follow the steps present in "Import" tab present the sidenav of this documentation.

Below are the input fields used in this screen:

Name of the field Entry Mode Mandatory Comments
Pincode Number Yes You have to fill this yourself.
Country Name Free Text Yes you can fill it yourself.
Country ISO Code None No System generated once you select country Name.
State Name Free text Yes You can fill it yourself.
City Name Free Text Yes You can fill it yourself.
Region Free text Yes You can fill it yourself.
Status Selection Yes System generated once you select partner name. Also, you can fill it yourself. Status could have following values - Active, Inactive
