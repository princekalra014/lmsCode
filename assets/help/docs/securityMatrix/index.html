Security matrix

Data given on this page tells us that which role can use specific function of specific module.

Roles

Given below is name of roles and defination of these roles.

Role's Name Defination
GSO(Ground Staff Officer) "GSO is lounge(outlet) staff login for accessing the WAS Client for the Operations.This is lounge staff login for accessing the WAS Client for the Operations.Login is specific to each outlet. The clients and services entitled for each outlet are controlled by LMS Admin side (i.e. “LMS BOS”)"
Lounge Manager Lounge Manager login for accessing the WAS Client. Lounge Manager has the access to void the transaction and to readmit the customer
Local Sales -AI creator This login is used by local sales team to create AIs for local clients. Only local outlets and regions could be selected (e.g. Malaysia local sales login can select Malaysia and Singapore outlets only
Region Sales-AI creator "This login is for regional clients and AIs creation, used by HK Sales Team. All outlet locations could be selected"
Finance This login is used for finance department for Reconciliation purpose,correct billing reports and to see transactional data.
BUH (Business Unit head) -Approver This login is used to approve/reject submitted Admission Instructions.
Admin (Super user) This login is used for giving user's permissions(roles)/outlet access making user inactive if (needed).

Outlet

An outlet is a physical location which is used to provide various services and facilities for the guests. Each outlet has the staff, equipment, and features that provide a set of services to the guests. The outlets can be owned by plaza group and operated by plaza group also, or they could be owned by a third party and operated by Plaza group using its processes, systems and staff.

The token is presented at the outlet, and the staff keys in the token to check and validate the entitlements for the guests based on the token. When a new partner is on board they provide a token list and details to the sales staff on how to identify the guests.

NOTE if the token is in DRAFT state then its editable, once approved non-editable. No No Yes Yes No Yes No
Generate AI Able to download the report in pdf format,Hollistic view of the Admission Instruction entitements, BIN range and it is applicable to what all outlets. No No Yes Yes No Yes No
